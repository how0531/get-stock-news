# 入門課：《散戶新聞實戰課——用對新聞，讓看盤效率翻倍》

> 零程式門檻的入門版課程。與技術版《用 Python 打造你的股市新聞情報系統》（見 `course/`）構成
> **入門 → 進階** 的雙產品線：入門課教「方法論的手動版」，技術課教「同一套方法論的自動化版」。

## 定位

| 項目 | 內容 |
|------|------|
| 目標學員 | 散戶小白：會看盤、會滑新聞 App，但完全不寫程式 |
| 核心承諾 | 每天滑一小時新聞 → 30 分鐘看懂重點；不再看到利多就追高 |
| 形式 | 6 單元教學投影片（`slides.html`）＋ 四個實戰練習（單元二～五） |
| 建議售價 | NT$690–990（入門引流品）或作為技術課的贈品/前導 |
| 與技術課的關係 | 入門課學員的自然升級路徑＝技術課（投影片第 36 頁內建升級鉤子） |

## 方法論對照（本課手動版 ⇄ 技術課自動版）

| 觀念 | 入門課（手動） | 技術課（自動） |
|------|----------------|----------------|
| 多來源蒐集 | 三個固定來源人工掃 | 10+ 家媒體 + 官方 OpenAPI 自動抓 |
| 轉載去重 | 肉眼認出「同一件事」 | 兩層去重演算法（norm_title + 內文指紋） |
| 熱度 | 手填熱度紀錄表、跟自己常態比 | quick_heat 自動評分 + z-score 異常偵測 |
| 落後指標/PIT | 「看到時價格已反應」的觀念 | actionable_ts 的 PIT 儲存實作 |
| 反向因子 | 「連推三天要小心」的現象認知 | expert_repeat 反向因子量化 |
| 事後驗證 | 手記 T+1/T+5 走勢 | T+1/T+5/T+20 報酬自動驗證 |

## 法遵注意

- 全課不含任何個股買賣建議；個股僅為教學示例（投影片封面、結尾與高風險頁均有免責聲明）
- 單元四「連推三天反向」以「市場老手的經驗觀察」呈現、標明「三天」非研究數字並附提醒，不構成操作建議
- 授課與行銷素材沿用 `course/operations/sales-ops-and-compliance.md` 的紅線規範
