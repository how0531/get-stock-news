# 模組 1｜導論與環境建置（講師教學筆記）

> 總長約 60 分鐘 / 6 堂。目標：讓「所有人」跑起第一次抓取——這模組決定退費率，寧慢勿快。

## 教學目標

- 學員理解整條管線的全貌（抓取 → 儲存 → 個股辨識 → 熱度）
- 建立正確心態：新聞是落後指標、本課教技術不教明牌
- 理解著作權與爬蟲禮儀的界線
- Windows/Mac 都能成功執行 `python scripts/main.py`

## 1-2 為什麼散戶需要自己的新聞系統（10'）

**講稿大綱**
1. 資訊鏈：公司公告（源頭）→ 通訊社 → 各媒體轉載 → 新聞 App → 你。每層都在加延遲
2. 兩個關鍵觀念（之後模組會反覆出現）：
   - **新聞是落後指標**——正確用法是「量化關注度的變化」，不是「看到利多就追」
   - **邊際變化 > 絕對熱度**——台積電每天都有 30 則新聞不稀奇；小型股突然從 0 變 8 則才是訊號
3. 法人為什麼強：不是消息比較神，是**處理資訊的系統**比較好。這門課就是幫你蓋那個系統

## 1-3 法遵與著作權（12'）⚠️ 必錄，不可跳過

**講稿大綱**
1. **著作權**：新聞「事實」本身不受保護，但「文字表達」受保護
   - 我們的系統：抓標題 + 連結 + 發布時間做量化分析 → 低風險
   - 全文只做個人研究用途的內部處理，**絕不重製全文對外散布**（repo README 也是這樣聲明的）
2. **爬蟲禮儀**：
   - 課程用的來源以「公開 API / RSS / sitemap」為主——這些是網站主動提供機器讀取的介面
   - 頻率節制：程式內建 `time.sleep`，盤中輪詢間隔 60 秒起跳
   - 不繞過付費牆、不破解反爬蟲
3. **投資法遵**：本課程內容為程式技術教學，非證券投資分析或投資建議；課程中出現的個股僅為程式輸出範例

## 1-4 環境建置 Windows（12'）

**Demo 步驟**（全程錄屏，不剪快轉——學員就是會卡在這裡）
1. 安裝 Python 3.11+（python.org，勾選 Add to PATH）
2. 安裝 VS Code + Python 擴充
3. `git clone`（或直接下載 zip——對零基礎學員 zip 更友善）
4. 建 venv 並安裝依賴：
   ```powershell
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt
   ```
5. Windows 特別提醒：執行任何 script 前設 `PYTHONUTF8=1`（示範 `$env:PYTHONUTF8=1`），否則中文輸出會亂碼

**常見卡關**（做成講義附錄）
- `python` 不是內部或外部命令 → PATH 沒勾
- pip 裝 pyarrow 失敗 → Python 版本太舊
- 亂碼 → 沒設 PYTHONUTF8

## 1-5 環境建置 Mac + 專案導覽（10'）

1. Mac：`brew install python@3.11` → 同上流程（`source .venv/bin/activate`）
2. 專案導覽（口述地圖，別逐檔講）：
   - `scripts/` 所有程式、`data/` 所有產出、`skills/` 兩份系統說明書
   - 「`data/` 目錄現在幾乎是空的，上完課它會被你自己抓的資料填滿」
