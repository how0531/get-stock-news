# 行銷計畫（Marketing Plan）

## 1. 行銷漏斗總覽

```
認知（Awareness）        社群短文/短影音、投顧既有頻道曝光
   ↓
興趣（Interest）         免費試看 3 堂、免費直播「10 分鐘抓下今天所有台股新聞」
   ↓
名單（Lead）             直播報名/講義下載 → 收 Email + LINE
   ↓
轉換（Conversion）       早鳥 14 天 + EDM 三封信 + 投顧會員專屬碼
   ↓
擴散（Referral）         學員作品展示、兩人同行價、結業證書分享
```

## 3. 通路策略

| 通路 | 角色 | 內容 |
|------|------|------|
| 合作投顧（會員 EDM/LINE/講座） | **主力導流**，預估 40–50% 銷量 | 會員專屬價 NT$2,380 + 專屬折扣碼（歸因用） |
| Facebook 社團（Python 台灣、量化交易、股癌相關社群） | 有機觸及 | 技術乾貨文（非硬廣），文末附試看連結 |
| Threads / X | 短內容 | 「開發者工具找 API」30 秒短影音最適合 |
| YouTube | 長尾 SEO | 免費直播回放 + 3 支教學短片（標題埋「台股新聞 API」「Python 爬蟲 股票」關鍵字） |
| 部落格/Medium | SEO 長尾 | 2 篇技術文：〈鉅亨網其實有公開 API〉〈什麼是 PIT，為什麼你的回測在騙你〉 |

**廣告預算（可選）**：首波建議 NT$30,000–50,000 測 FB/IG 廣告，受眾＝Python/投資交集興趣；ROAS < 2 即停，靠有機與投顧導流為主。

## 4. 投顧合作方案（談判框架）

> 實際條件依雙方合約為準，以下為建議起點。

| 項目 | 建議內容 |
|------|----------|
| 分潤 | 透過投顧折扣碼成交：投顧抽 20–30%；投顧主動辦講座導購：抽 30–40% |
| 會員專屬價 | NT$2,380（比早鳥高、比定價低——不打壞早鳥急迫感，又給會員專屬感） |
| 投顧得到什麼 | 會員加值服務（提升續訂）、監製掛名、B 端授課選項（見 §6） |
| 我方得到什麼 | 精準受眾導流、專業背書、見證素材 |
| 法遵紅線 | 課程文案與內容**不得**由投顧名義提供個股建議連動（詳見 operations 法遵文件）；監製掛名文案需經投顧法遵審閱 |
| 歸因機制 | 專屬折扣碼 + UTM 連結，月結報表 |

3. **免費試看 3 堂**：1-1 課程地圖、2-1 找 API、7-1 熱度定義
